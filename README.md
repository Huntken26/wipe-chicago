# wipe-chicago

## OVERVIEW

This project is a Full Stack Application for the cleaning business- Wipe Chicago LLC. This is my first professional build and I am using the skills I've learned in the Coding Bootcamp to develop the Front and Back end. Wipe Chicago wants a web app where clients can check their schedule, book appointments and communicate with workers in case of emergencies. There wasn't a mockup or general idea given so I am developing the full concept myself- logos, typography, branding etc. I will be using Canva, HTML, CSS, Bulma CSS framework, Javascript, fontawesome, google fonts among other technologies. I will also add to this project as I gain more skills in my Full Stack Bootcamp.
